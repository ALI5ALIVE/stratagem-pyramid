

# CoAnalyst Deck: Reframe from Category Creation to Intelligence Layer Positioning

## Context

The CoAnalyst deck currently frames CoAnalyst as defining a **new product category** ("Aviation Data Intelligence"). Since the category is already defined as **Operational Performance Platform**, the deck should instead position CoAnalyst as the **intelligence layer** that sits above the three core modules (Safety, Content, Training Manager 365) and powers the DTOP model.
